# antennas

This directory contains YAML files describing the coaxial cable connections for aircraft antennas, including:

- COM Antenna (VHF communications)
- XPDR Antenna (Transponder/ADS-B)
- GA 35 Antenna (GPS/WAAS)

Each file documents connector types, pinouts, cable types (typically RG400 coax), and references to installation manuals for proper mounting and wiring practices.
